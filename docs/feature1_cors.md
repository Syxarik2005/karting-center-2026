# Feature 1: CORS Middleware

**Симптом/Цель:**
Для тестирования API из локального web-песочницы (sandbox) или других web-клиентов запросы блокировались политикой CORS браузера.

**Требования:**
Разрешить кросс-доменные запросы (CORS) для всех источников в режиме разработки.

**Реализовано:**
В `cmd/api/main.go` написан и добавлен middleware `corsMiddleware`, добавляющий заголовки `Access-Control-Allow-Origin: *` и перехватывающий `OPTIONS` запросы.

**Отправленные промпты:**
*(Пользователь)*: "Открой приложение чтобы я потыкал. Исравить 3 разных бага и написать 3 фичи"

**Ручная проверка:**
Вызовы API из `app_sandbox.html`, запущенного на порту `8081`, проходят успешно к API на `8080`.
