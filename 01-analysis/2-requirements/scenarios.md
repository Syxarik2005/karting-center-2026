# Основные сценарии — Клиентское приложение «Апекс»

> Формат: Given / When / Then (BDD-стиль).  
> Привязка к User Stories из `user-stories.md`.

---

## SC-01 · Авторизация нового клиента (US-01)

| | |
|---|---|
| **Given** | Клиент впервые открывает приложение |
| **When** | Вводит номер телефона +7 (999) 123-45-67 и нажимает «Получить код» |
| **Then** | На экран ввода кода отображается поле для 4-х цифр, на телефон приходит OTP |
| **When** | Вводит корректный код (0000 — тестовый) |
| **Then** | Создаётся новый аккаунт, клиент попадает на главный экран SCR-001 |

### Альтернативный сценарий: неверный код
| | |
|---|---|
| **When** | Клиент вводит неверный OTP (1234) |
| **Then** | Ошибка «Неверный код», возможность повторить ввод или запросить новый код |

## SC-02 · Просмотр расписания с фильтрацией (US-02, US-03)

| | |
|---|---|
| **Given** | Клиент авторизован, сегодня 04.07.2026, в расписании слоты на 05.07, 08.07 и 15.07 |
| **When** | Открывает главный экран SCR-001 без фильтров |
| **Then** | Видит слоты на 05.07 и 08.07 (ближайшие 7 дней), слот на 15.07 не показан |
| **When** | Устанавливает фильтр «Конфигурация: LONG» |
| **Then** | В списке остаются только слоты с длинной трассой |
| **When** | Расширяет диапазон дат до 20.07 |
| **Then** | Появляется слот на 15.07 |

### Альтернативный сценарий: нет доступных слотов
| | |
|---|---|
| **Given** | На ближайшие 7 дней нет запланированных заездов |
| **When** | Клиент открывает SCR-001 |
| **Then** | Показан empty state: «Пока нет доступных заездов» |

## SC-03 · Успешное бронирование заезда (US-04)

| | |
|---|---|
| **Given** | Слот на 06.07 14:00, трасса SHORT, маршал Алексей, свободно 5 из 8 картов |
| **When** | Клиент тапает на слот, переходит на SCR-002 |
| **Then** | Видит полную информацию: время, трасса, маршал (рейтинг 4.8★), место сбора, выбор экипировки |
| **When** | Выбирает «Своя экипировка» и нажимает «Записаться» |
| **Then** | Индикатор загрузки → бэкенд возвращает 201 → переход на SCR-004 (детали бронирования, статус ACTIVE) |
| **Then** | Свободных картов на слоте стало 4 из 8 |

---

## SC-04 · Бронирование — конфликт (нет мест) (US-04)

| | |
|---|---|
| **Given** | На слоте осталось 1 свободное место |
| **When** | Два клиента одновременно нажимают «Записаться» |
| **Then** | Первый получает 201 Created — бронь создана |
| **Then** | Второй получает 409 Conflict — алерт «Места закончились», кнопка «Записаться» блокируется |
| **Then** | Приложение второго клиента обновляет данные слота (available_karts = 0) |

---

## SC-05 · Бронирование с прокатной экипировкой (US-04)

| | |
|---|---|
| **Given** | На слоте есть свободная прокатная экипировка |
| **When** | Клиент выбирает «Прокат» |
| **Then** | Отображается стоимость проката (rental_tariff из API), кнопка «Записаться» доступна |
| **When** | Прокатного фонда больше нет |
| **Then** | Опция «Прокат» заблокирована с пояснением «Прокатная экипировка закончилась» |

---

## SC-06 · Отмена бронирования клиентом — заблаговременная (US-06)

| | |
|---|---|
| **Given** | У клиента активная бронь на заезд через 5 часов |
| **When** | Нажимает «Отменить запись» на SCR-004 |
| **Then** | Попап подтверждения: «Вы уверены? Место будет освобождено.» |
| **When** | Подтверждает отмену |
| **Then** | Бэкенд возвращает 200 OK, статус → CANCELLED_BY_CLIENT, место освобождается |

---

## SC-07 · Отмена бронирования клиентом — слишком поздно (US-06)

| | |
|---|---|
| **Given** | У клиента активная бронь на заезд через 30 минут |
| **When** | Нажимает «Отменить запись» |
| **Then** | Бэкенд возвращает 400 Bad Request |
| **Then** | Показан алерт: «Отмена невозможна: до заезда осталось менее 2 часов» |
| **Then** | Бронь остаётся в статусе ACTIVE |

---

## SC-08 · Отмена заезда центром по погоде (US-09)

| | |
|---|---|
| **Given** | Клиент имеет активную бронь на слот, центр отменяет слот из-за грозы |
| **When** | Клиент открывает приложение |
| **Then** | Бронь отображается со статусом «Отменён центром» и причиной «Неблагоприятные погодные условия» |
| **Then** | Push-уведомление: «Ваш заезд 06.07 в 14:00 отменён из-за непогоды» |
| **When** | Другой клиент пытается записаться на этот же отменённый слот |
| **Then** | Запись невозможна, ошибка «Заезд отменён» |

---

## SC-09 · Оценка маршала (US-07)

| | |
|---|---|
| **Given** | Заезд клиента завершён (статус COMPLETED), маршал ещё не оценён |
| **When** | Клиент тапает «Оценить маршала» на SCR-004 |
| **Then** | Открывается SCR-005 (модальное окно) с именем маршала и шкалой 1-5 |
| **When** | Ставит 5 звёзд, пишет «Отличный инструктор!» и нажимает «Отправить» |
| **Then** | Оценка сохранена, кнопка «Оценить» больше не отображается |

### Альтернативный сценарий: повторная оценка
| | |
|---|---|
| **When** | Клиент пытается оценить того же маршала за тот же заезд |
| **Then** | Кнопка отсутствует / запрос отклонён |

---

## SC-10 · Просмотр профиля и выход (US-08)

| | |
|---|---|
| **Given** | Клиент авторизован |
| **When** | Переходит на экран профиля SCR-006 |
| **Then** | Видит: имя «Иван», телефон «+7 (999) 123-45-67», статус «Постоянный клиент» (если есть) |
| **When** | Нажимает «Выйти» |
| **Then** | JWT-токен удалён, клиент перенаправлен на экран авторизации |
