# Бриф на дизайн — Клиентское приложение «Апекс»

## 1. О продукте

**Тип приложения:** Мобильное клиентское приложение (iOS/Android) для записи на заезды в картинг-центре «Апекс».

**Целевая аудитория:**
- Молодые люди 16–35 лет, активный досуг
- Семьи с детьми (новичковые трассы)
- Постоянные клиенты (корпоративные выезды, соревнования)

**Основные сценарии использования:**
1. Выбор заезда по расписанию (дата, трасса, маршал)
2. Бронирование места с выбором экипировки
3. Управление бронированиями (отмена, просмотр)
4. Оценка маршала после заезда

## 2. Визуальная концепция

### Настроение (Mood)
- **Скорость и адреналин** — чувство динамики
- **Современность** — чистый и технологичный интерфейс
- **Уличная атмосфера** — ощущение открытого пространства

### Цветовая палитра

| Цвет | HEX | Назначение |
|---|---|---|
| Основной (Racing Red) | `#E53935` | Акценты, CTA-кнопки |
| Тёмный фон | `#1A1A2E` | Фон основных экранов |
| Вторичный тёмный | `#16213E` | Карточки, блоки |
| Акцент (Neon Yellow) | `#FFD600` | Предупреждения, статусы |
| Текст основной | `#FFFFFF` | На тёмном фоне |
| Текст вторичный | `#B0BEC5` | Пояснения, подписи |
| Успех (Green) | `#00E676` | Подтверждение бронирования |
| Ошибка (Orange) | `#FF6D00` | Ошибки, отмены |

### Типографика
- **Заголовки:** Inter Bold, 20–24px
- **Основной текст:** Inter Regular, 14–16px
- **Кнопки:** Inter SemiBold, 16px, UPPERCASE

### Иконки и графика
- Стиль: outlined, rounded
- Библиотека: Material Icons или Lucide
- Иллюстрации: минималистичные, line art (карт на трассе, шлем, флажок)

## 3. Компоненты UI-кита

### Карточка слота
- Тёмный фон `#16213E` с лёгким `border-radius: 16px`
- Левая полоса цвета конфигурации трассы (RED = LONG, YELLOW = SHORT)
- Информация: время, маршал (аватар + рейтинг), свободно X/Y картов
- Micro-animation: пульсация при `available_karts <= 2`

### Кнопка «Записаться»
- Фон: градиент `#E53935 → #D32F2F`
- Текст: белый, SemiBold, UPPERCASE
- Состояние disabled: opacity 0.4
- Состояние loading: спиннер

### Статус-бейджи
- `ACTIVE` — зелёный бейдж
- `CANCELLED_BY_CLIENT` — серый
- `CANCELLED_BY_CENTER` — оранжевый + иконка «⚠️»
- `COMPLETED` — синий

### Bottom Navigation
- 3 вкладки: Расписание | Мои заезды | Профиль
- Иконки: флажок | список | пользователь

## 4. Анимации и переходы
- Переход между экранами: slide left/right (300ms ease)
- Появление карточек: fade-in + slide-up (200ms stagger)
- Pull-to-refresh: стандартный системный
- Skeleton loading: пульсирующие placeholder'ы

## 5. Адаптивность
- Основная платформа: мобильные телефоны (375–428px)
- Ориентация: только портретная
- Без планшетной версии на MVP

## 6. Референсы
- Dark-themed спорт-приложения (Nike Run Club, Strava dark mode)
- Booking-приложения (Яндекс.Афиша, Профи.ру)
