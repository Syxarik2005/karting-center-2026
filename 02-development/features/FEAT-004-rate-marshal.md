# FEAT-004 — Оценка маршала (SCR-005)

## Цель
После завершённого заезда клиент может поставить маршалу оценку 1–5 звёзд с
необязательным комментарием — это единственный сигнал качества, который есть
у владельца («у меня есть пара ребят-звёзд… хочу видеть, кто как заходит людям»).

## Требования
- `customer-brief.md` — «чтобы после заезда клиент мог поставить оценку маршалу».
- `domain-description.md` — открытый вопрос №4 (обязательна ли оценка при
  следующем входе, или добровольна) — решили в пользу **добровольно**, т.к.
  принудительный шаг перед использованием остального приложения — плохой UX
  и не был явно запрошен заказчиком; кнопка «Позже» есть.
- `screen-registry.md` — SCR-005 (модальное окно/bottom sheet, звёзды 1–5,
  опциональный комментарий).
- `openapi.yaml` — `POST /client/bookings/{id}/rate`.

## Реализация
- `src/pages/RateMarshalPage.tsx` — рендерится как bottom sheet (маршрут
  `/bookings/:id/rate`, в `App.tsx` для этого пути скрыт нижний таб-бар).
- `src/components/StarRating.tsx` — доступный виджет (`role="radiogroup"`,
  `aria-checked`), а не просто пять кликабельных <span>.
- Кнопка «Отправить» задизейблена, пока `rating === 0` — нельзя отправить
  пустую оценку. Кнопка «Позже» просто уводит назад без вызова API.
- В `BookingDetailsPage` кнопка «Оценить маршала» показывается только когда
  `status === 'COMPLETED' && !client_rating` — то есть не показывается ни для
  активных броней, ни повторно после того как оценку уже поставили.

## Промпты (переданные ИИ-ассистенту)
```
1. "SCR-005 — bottom sheet с фото/именем маршала, звёзды 1-5, комментарий
    опционален, кнопка отправки задизейблена при rating=0. Добавь кнопку
    "Позже" — оценка не обязательна (см. открытый вопрос №4 в domain-description)."

2. "Кнопка "Оценить маршала" на SCR-004 должна появляться только для
    завершённых броней и пропадать после того, как оценка отправлена —
    добавь client_rating в Booking как чисто фронтовое поле."
```

## Ручная проверка
1. Открыть `/bookings` → вкладка «Прошедшие» → `booking-2` (COMPLETED,
   без оценки) → должна быть видна кнопка «Оценить маршала».
2. Открыть `/bookings/booking-2/rate` → нижний таб-бар должен исчезнуть
   (это модальный шаг, а не обычный экран).
3. Кнопка «Отправить» задизейблена, пока не выбрана хотя бы 1 звезда.
4. Поставить 5 звёзд + комментарий → отправить → редирект обратно на детали
   брони, кнопка «Оценить маршала» больше не показывается.
5. Нажать «Позже» без выбора звёзд → должен просто закрыться шит без ошибок
   и без вызова `rateMarshal`.
